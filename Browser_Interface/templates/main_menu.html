Bus timetable processor

Welcome to the bus timetable processor. This application is designed to create bus schedules directly from timetables in JDF format.

Getting started

Default scheduling

In Upload JDF, select a folder with JDF 1.11 timetable batch to process and upload to a server.

In Query trips, you can select a time range to verify trips in the uploaded batch.

Use Schedule buses to create a schedule for buses which serve all trips in the uploaded JDF within 24-hour window.

Human-readable timetables

You can download a human-readable version on the timetables on the page Export timetables to XLSX

You can edit the downloaded timetables in external spreadsheet processor and then upload them using Import timetables in XLSX.
After you get familiar with the timetable format, you can try importing your own.

Using map data

Use List of stops to add new stop locations to the server. These can also be used for more accurate bus scheduling.

Use Visualize schedules to create plots on map background to visualize the schedules.
